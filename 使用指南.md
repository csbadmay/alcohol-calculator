# 酒精配比计算器 - 使用指南

## 这个工具现在能做什么

它主要服务三类现场需求：

- 温度换算：酒精计 + 温度计
- 混合计算：不同酒度、不同重量或体积的酒混合
- 加水降度：高酒度原酒加水降到目标酒度

系统内部会结合：

- **表 B.1** 做温度换算
- **表 A.1** 做密度、重量和体积换算

## 适合的输入方式

- 温度支持小数，例如 `24.5`
- 酒精计读数支持小数，例如 `10.8`
- 系统会对非表格点自动插值，不需要你手工查表
- 混合和加水时，数量支持输入公斤或升

## 如何在电脑上打开

1. 在项目目录运行：`python -m http.server 8000`
2. 浏览器访问：`http://localhost:8000`

## 如何在 iPhone 上使用

1. 打开部署后的 GitHub Pages 地址
2. 在 iPhone 的 Safari 中访问
3. 点击“分享” -> “添加到主屏幕”

## 使用建议

- 输入前先确认酒样已经按你的流程完成蒸馏
- 如果输入超出表 B.1 支持范围，页面会直接提示无法换算
- 页面显示为两位小数，内部计算精度高于显示精度
- 目标反推默认给出两种原酒各自需要的重量和体积，方便你按现场实际条件取值
