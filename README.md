# 酒精配比计算器

基于 **GB 5009.225-2023 表 A.1 / 表 B.1** 的手机优先换算工具。

当前支持：

- 温度换算：测量温度 + 酒精计读数 -> 20℃ 标准酒精度
- 重量体积换算：已知酒度和重量或体积，换算另一种单位
- 混合结果：多种不同酒度、不同单位的酒混合后，求最终酒度
- 目标反推：想得到指定重量或体积、指定酒度的成品，反推两种原酒的用量
- 加水降度：已知原酒数量和目标酒度，计算需要加多少水

温度换算使用完整表 B.1 数据，并对非表格点进行插值。重量体积、混合和加水相关计算使用表 A.1 密度数据。

## 本地使用

1. 在项目目录启动静态服务：`python -m http.server 8000`
2. 浏览器打开：`http://localhost:8000`

## iPhone 使用

推荐直接使用 GitHub Pages 部署后的公网地址，在 Safari 中打开后添加到主屏幕。

## 校验

运行计算核心测试：

```bash
node calculator.test.js
```

## 项目文件

- `index.html`: 多功能手机界面
- `calculator.js`: 温度换算、密度、混合和降度计算核心
- `app.js`: 表单交互
- `data.js`: 表 A.1 / 表 B.1 数据
- `derived-grid.js`: 温度换算 0.1 网格派生数据
