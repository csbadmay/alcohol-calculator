# 酒精计温度换算器

基于 **GB 5009.225-2023 表 B.1** 的手机优先换算工具。

它只做一件事：

- 输入测量温度
- 输入酒精计读数
- 输出 **20℃ 标准酒精度**

内部使用完整表 B.1 数据，并对非表格点进行插值，适合现场输入如 `24.5℃ / 10.8%vol` 这类小数值。

## 本地使用

1. 在项目目录启动静态服务：`python -m http.server 8000`
2. 浏览器打开：`http://localhost:8000`

## iPhone 使用

1. 电脑和 iPhone 连接同一 Wi-Fi
2. 电脑运行静态服务
3. 通过 `ipconfig` 查看电脑 IPv4 地址
4. Safari 打开 `http://[电脑IP]:8000`
5. 点击分享，选择“添加到主屏幕”

## 校验

运行计算核心测试：

```bash
node calculator.test.js
```

## 项目文件

- `index.html`: 单页手机界面
- `calculator.js`: 表 B.1 插值计算核心
- `app.js`: 表单交互
- `data.js`: 表 B.1 数据
